## 值传递和引用传递

### 值传递

指调用函数时，**将实际参数复制一份**传递到函数中，对形参的修改**不会影响实际参数**

### 引用传递

在调用函数时，将**实际参数的地址**直接传递到函数中，对形参的修改会直接影响到实际函数

> 其实就和C++中传指针一个意思

### 为什么说Java只有值传递？

Java会将对象地址的拷贝传递给被调函数作为形参

绝对不能认为Java中有引用传递
